# Menu Editing Guide

Edit only this file to change the menu:

`assets/js/menu-data.js`

## What you can change there
- hero text on the menu page
- pricing strip
- signature cocktails
- pricing tiers
- classics
- beer / wine / zero-proof
- live notes / announcements

## Typical workflow
1. Open `assets/js/menu-data.js`
2. Change names, descriptions, prices, or section text
3. Save
4. Push to GitHub

## Push commands
```bash
git add .
git commit -m "Update menu"
git push
```
